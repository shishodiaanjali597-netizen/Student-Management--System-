# Student Management System

A backend project built with Java + JDBC + MySQL

## Features
- Add student
- View all students
- Update student
- Delete student
- Marks management
- Attendance tracking
- Fee management

## Database Tables
- students
- marks
- attendance
- fees
- admin

## Tech Stack
- Java
- JDBC
- MySQL
- Spring Boot (coming soon)
- AWS (coming soon)

## Project Status
🚧 In Progress

## Setup
1. Install MySQL
2. Run database.sql
3. Update password in DatabaseConnection.java
4. Run Main.java
